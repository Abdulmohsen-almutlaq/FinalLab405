# Lab Final 405


Web app to store and manage bookmarks.

##  Overview

React + PHP + PostgreSQL app.

##  Demo
[![Watch the video](https://img.youtube.com/vi/UkkaJjSIh8Y/0.jpg)](https://www.youtube.com/watch?v=UkkaJjSIh8Y)



##  Features

- CRUD API 
- React UI

##  Setup

### Backend
```bash
cd Backend
docker-compose up -d
```

### Frontend
```bash
cd Bookmark
npm install
npm run dev
```
